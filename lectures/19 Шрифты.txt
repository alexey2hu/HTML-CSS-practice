Что есть:
	0. Основные форматы шрифтов
	1. Подключения шрифтов

(0) - Основные форматы шрифтов:

    EOT (Поддерживается в Internet explorer)
    WOFF (поддерживается всеми основными браузерами)
    WOFF2 (на 30% легче - быстрее подгрузка шрифтов)
    TTF (Лучше не использовать, а конвертиовать)

(1) - Подключения шрифтов:
    
    Подключение ссылкой:
        <!-- Ссылкой через google fonts (HTML) -->
        <link rel="preconnect" href="https://fonts.googleapis.com">
        <link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
        <link href="https://fonts.googleapis.com/css2?family=Montserrat:ital,wght@0,100..900;1,100..900&display=swap" rel="stylesheet">

        /* Ссылкой через google fonts (CSS) */
        @import url('https://fonts.googleapis.com/css2?family=Montserrat:ital,wght@0,100..900;1,100..900&display=swap');

        /* <weight>: Use a value from 100 to 900 */
        /* <uniquifier>: Use a unique and descriptive class name */
        .montserrat-<uniquifier> {
            font-family: "Montserrat", sans-serif;
            font-optical-sizing: auto;
            font-weight: <weight>;
            font-style: normal;
        }

    Подключение загруженных шрифтов:
        @font-face {
            font-family: "FontName";
            src:
                url("/path/to/fonts/example-font.woff") format("woff"),
                url("/path/to/fonts/example-font.eot") format("embedded-opentype");
            font-weight: normal;
            font-style: normal;
            font-display: fallback;
        }

    Универсальный оптимизированный вариант:
        /* От быстрого к медленному при отсутствии возможности подключить */
        @font-face {
            font-family: "FontName";
            font-weight: 400;
            font-style: normal;
            src: 
            local(Font Name 1),
            local(Font-Name-alternative),
            url("/path/to/fonts/example-font.woff2") format("woff2"),
            url("/path/to/fonts/example-font.woff") format("woff"),

            /* Нет необходимости: (IE8-10: < 0.5% пользователей) +Старый IE воспринимает только перый шрифт */
            url("/path/to/fonts/example-font.eot") format("eot");
            
            /* swap быстрее всех!!! */
            font-display: swap;
        }

        .el {
            font-family: "FontName", sans-serif;
            font-weight: 800;
        }

(1) - Единицы измерения:
    Абсолютные(Не адаптивны, не подходят для людей с огр. возможностями):
        !- px (Пиксели, в них все автоматически переводит браузер)

    Относительные(Зависят от контекста, более адаптивны):

        /* Зависит от предыдущего значения в px родителей включая HTML */
        /* Берет в учет свойтсва всех родителей! (Стакается) */
            !- em (относительная единица измерения)
            (Size = (Parent px * em))

            .Parent {
                /* 16px * 1.5 = 24px */
                font-size: 1.5em;
            }

            .Child {
                /*!!! 24px * 1.5 = 36px */
                font-size: 1.5em;
            }

        /* Берет в учет только свойтсва HTML как главного родители */
            !- rem (root em)
            (Size = (HTML px * rem))

        /* % от размера окна браузера (1 vh или vw = 1% viewport) */
            !- vh (viewport height)
            (Size = (viewport height / 100 * vh))
            
            !- vw (viewport width)
            (Size = (viewport width / 100 * vw))

        /* vh/vw но влияющий параметр выбирается выстой или шириной в зависимости от выполнения условия */
            /* 1% от меньшего из размеров окна просмотра — ширины или высоты */
            !- vmin (viewport minimum)

            /* 1% от большего из размеров окна просмотра — ширины или высоты */
            !- vmax (viewport maximum)
            
        
(2) - Какие есть проценты:
    /* % height (Высота, % от родителя) */
    /* % width (Ширина, % от родителя) */
    /* % position (top, left (Отступы, % от родителя)) */
    /* % margin и padding (Отступы внутри родителя, % по ширине родителя) */
    /* % translate (Отступ по x или y, % от высоты или ширины элемента) */
    /* % font-size и line-height (Размер шрифта и высота строки, % от главного родителя) */

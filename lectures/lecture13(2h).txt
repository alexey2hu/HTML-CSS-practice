Что такое CSS:
	CSS (Cascading Style Sheets, «каскадные таблицы стилей») — язык стилей, с помощью которого оформляют веб-страницы.

Особенности каскадности CSS:
	/* 
		Наследование свойств от Parent к Child элементам
		Не все свойства наследуются, например
		color - наследуется, а border нет.
	*/

Селекторы CSS:
	/* 
		Селектор {
			/* Строчка целиком называется ПАРАМЕТРОМ */
			свойство: значение;
			padding: 10px;
			border: 1px solid #333;
		}
	*/
	/* Обращение ко всем тегам div */
		div {}
	/* Обращение к определенному классу (.className {}) */
		.clss {}
	/* Обращение по id (#ID {}) */
		#ID {}
	
	/* Обращение к нескольким классам (.clss1, .clss2 {}) */
		.clss1, .clss2, ... {}

	/* Обращение к селектору с определенным классом
		(div.className {}) */
		div.clss {}

	/* (Пробел) Обращение к ДОЧЕРНЕМУ ЭЛЕМЕНТУ по его классу внутри определенного селектора (div .className {}) */
		div .clss {}

	/* (>) Обращается только к вложенным элементам внутри первого параметра на первом уровне вложенности и не смотрит дальше (div>.className {}) */
		.parentClass>.childClasses {}

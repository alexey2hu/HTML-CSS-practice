Что есть:
    0. Отличие псевдоэлементов от псевдоклассов
    1. Псевдоэлементы (Создание)
    2. Псевдоэлементы (Изменение частей элементов)
    3. Примеры использования

(0) - Отличие псевдоэлементов от псевдоклассов:
    /* Псевдоклассы - ':'
        (Работает с уже существующими элементами)
        (Позволяют применять один и тот же набор стилей к нескольким элементам) */
    
    /* Псевдоэлементы - '::'
        (Создает новые элементы)
        (Позволяют стилизовать определённые части элемента.) */


(1) - Псевдоэлементы (Создание):

    /* Создает новый элемент ДО всех child элементов */
    .block::before {
        /* Обязательное свойство */
        content: "text";

        /* (def: inline) */
        display: block;
    }

    /* Создает новый элемент ПОСЛЕ всех child элементов */
    .block::after {
        content: "text";
        display: block;

        /* Используется в большинстве случаев */
        position: absolute;
        top: 200px;
        left: 12.5rem;
    }

(2) - Псевдоэлементы (Изменение частей элементов):
    /* Изменение ПЕРВОЙ СТРОКИ в тексте */
    .text::first-line {}

    /* Изменение ПЕРВОЙ БУКВЫ в тексте */
    .text::first-letter {}

(3) - Примеры использования:

    /* Декор перед каждым элементом в списке */
    .list-item::before {
        content: "";
        display: inline-block;

        margin-right: 1rem;

        width: 1rem;
        height: 1rem;

        background-color: green:
        boreder-radius: 50%;
    }

    /* Добавление картинки в кнопку */
    .btn::after {
        /* Если не подходит формат изображения оставить пустым */
        content: url("link");

        display: inline-block;

        margin-left: 1rem;

        width: 1em;
        height: 1em;

        background: url("link") 0 0 / 100% 100%;
    }

    /* Обращение к существующему псевдоэлементу с помощью псевдокласса (Цель: сменить изображение) */
    .btn:hover::after {
        background: url("new_link") 0 0 / 100% 100%;
    }

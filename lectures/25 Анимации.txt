Что есть:
    1. Свойство animation
    2. Анимирование - @keyframes

(1) - Свойство animation:
    .an{
        /* animation: name duration timing-function delay iteration-count direction fill-mode; */
        /* (!!!) Анимации можно комюинировать через запятую (!!!) */
        animation: move-block 2s ease-in infinite alternate,
        color-block 1s 2s forwards;
        /* Включает в себя 
            /* Имя */
            animation-name: name;
            /* Время */
            animation-duration: 0s;
            /* Режим */
            animation-timing-function: ease;
            /* Повторения */
            animation-iteration-count: infinite;
            /* Направление и зацикливание */
            animation-direction: alternate/reverse;
            /* пауза или продолжение при событиях */
            animation-play-state: paused;
            /* Задержка перед проигрываанием, при МИНУСОВОЙ начинается уже перемотанной вперед */
            animation-delay: -3s;
            /* Какой кадр до начала и после анимации (при forwards последний кадр остается при завершении) */
            animation-fill-mode: forwards;

        /* Доп. */
            /* Перезапуск анимации при событиях */
            animation-name: none
    }

(1) - Анимирование - @keyframes:
    /* Перемещение + изменение формы */
    @keyframes move-block {
        0% {
            bottom: 0%;
        }
        100% {
            bottom: calc(100% - 150px);
            border-radius: 50%;
        }
    }

    /* Изменение цвета навсегда (forwards) */
    @keyframes color-block {
        100% {
            background-color: crimson;
        }
    }

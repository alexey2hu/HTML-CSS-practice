Что есть:
	0. Что такое Специфичность
	1. Виды специфичности

Специфичность (вес селектора) - приоритет определения свойств для элемента

Порядковая специфичность css:
	/* Выберет последний заданный конфликтующий параметр */
	h1 {
		color: red;
	}
	h1 {
		color: green;
	}

Весовая специфичность:
	/* вес = 1 (элемент, псевдоэлемент) */
		h1 {}
	/* вес = 10 (классы, атрибуты, псевдоклассы) */
		.clss {}
		[clss] {}
	/* вес = 100 (ID) */
		#blockID {}

	/* вес = 1.000 (style элемента в HTML) */
		<h1 style='color:red; background:gray;'><h1>

	/* вес = 10.000 (!important) */
	/* Лучше к нему не прибегать */
		h1 {
			color: red !important;
		}
	
Сочетание специфичности:
	/* вес = (10 + 1) = 11 (Вес элемента + вес класса) */
		h1.clss {}

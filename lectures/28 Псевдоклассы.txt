Что есть:
    1. Псевдоклассы (Визуальные)
    2. Псевдоклассы (Обращения)

(1) - Псевдоклассы (Визуальные):

    /* Срабатывает если уже переходили по ссылке */
    .link:visited {}

    /* Срабатывает при наведении */
    .link:hover {}

    /* Срабатывает при нажатии */
    .link:active {}

    /* Срабатывает при фокусировке на элементе */
        .btn:focus {}

        /* (!!!) Так можно указать дефолтный эффект для всех элементов (!!!) */
        :focus {}

(2) - Псевдоклассы (Обращения):

    /* Обращение к ПЕРВОМУ дочернему элементу с указаным именем */
    list_item:first-child {}

    /* Обращение к ПОСЛЕДНЕМУ дочернему элементу с указаным именем */
    list_item:last-child {}

    /* Обращение к (3) по счету дочернему элементу с указаным именем */
    list_item:nth-child(3)

        /* Обращение ко всем (odd) НЕЧЕТНЫМ дочерним элементам с указаным именем */
        list_item:nth-child(odd)

        /* Обращение ко всем (even) ЧЕТНЫМ дочерним элементам с указаным именем */
        list_item:nth-child(even)

        /* Можно записывать в скобках даже логику для обращения */

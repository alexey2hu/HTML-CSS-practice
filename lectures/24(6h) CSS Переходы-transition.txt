    1. Свойство transition

(1) - Свойство transition:
    .tion {
        /* Анимация (к чему применить | сколько длится | задержка) */
        transition: border-radius 2s, background-color 1s 2s;

        /* Включает в себя (К чему применять) */
        transition-property: border-radius, background-color;
        /* (Время применения) */
        transition-duration: 2s, 1s;
        /* (Задержка перед началом анимации) */
        transition-delay: 1s, 0s;

        /* Режим анимации (def: ease) */
        transition-timing-function: ease;
        /* Ускорение в начале */
        transition-timing-function: ease-in;
        /* Ускорение в конце */
        transition-timing-function: ease-out;
        /* Гибкие настройки анимации */
        transition-timing-function: cubic-bezier(0.075, 0.82, 0.165, 1);
    }

Что такое HTML:
	HTML (HyperText Markup Language) — язык разметки для создания и структурирования веб-страниц.

Emmet команды:
	1. ! - Основа страницы
	2. link: - Подключение favicon, css и т.д.

Парные теги - это: <div {params}></div>,
Непарные теги - это: <meta {params}>

Примеры содержимого страницы:
	<!DOCTYPE html> <!-- Какую версию html будем использовать -->
	<html lang="ru"></html> <!-- Оболочка всего, язык страницы -->

	<head></head> <!-- Неотображаемая инф-ция страницы -->
		<meta charset="UTF-8"> <!-- Метаданные, кодировка страницы -->
		<meta name="viewport" content="width=device-width, initial-scale=1.0" /> <!-- Возможность адаптива -->
		<link rel="shortcut icon" href="favicon.ico" type="image/x-icon" />
		<!-- Тег подключения иконки -->
		<title>Название страницы</title>
	
	<body></body> <!-- Отображаемая инф-ция страницы -->
